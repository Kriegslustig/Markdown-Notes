Save Rename New Delete Viewmode
  • N: Create a note
  • S: Save the active note
  • D: Delete the active note
  • R: Rename a note
  • T: Toggle the tree
  • H: Toggle the help dialoge
  • V: Toggle the view-mode
  • E: Encrypts and save the current note
  • C: Decrypts the current note
    Markdown-Notes
